= <Rain Past Sky Azure Clouds Scattered Place> (<雨过天青云破处>) (~ CE 1120) (by: 赵 佶?) =

	云外雨初霁: 天青澈湛蓝.
+	Clouds outer rain first emerged: Sky azure clear through blue.
	雨过天青云破处:
+	Rain past sky azure clouds scattered place:
	这般颜色作将来.
+	This sort of colors be the future.

	抬眼望窗外: 窗外雨初霁.
+	Lift eyes viewing windows outer: Windows outer rain first submerged.
	日透层云霭: 天青色湛蓝.
+	Sun pierced dense cloudy mist: Sky azure color through blue.
	雨过天青云破处: 这般颜色作将来.
+	Rain past sky azure clouds scattered place: This sort of colors be the future.
